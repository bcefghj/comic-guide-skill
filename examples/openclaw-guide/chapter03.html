🤖
哆啦A梦
大雄，你知道我的百宝袋里有多少道具吗？
👦
大雄
好几千个？任意门、竹蜻蜓、时光包袱巾...

没错！小龙虾的「Skills 系统」就是它的百宝袋——目前已有 5,400+ 个技能，而且每天还在增加！
🎒
Skills = 百宝袋
每个 Skill 就是一个「道具」——一份 SKILL.md 文件，告诉小龙虾遇到什么情况、按什么步骤干活。不需要写代码，只需要会写 Markdown！
🛠️ 道具 ↔ Skill 对照表
🚪
≈ 任意门
url-to-markdown
抓取任何网页转 Markdown
🖼️
≈ 如果电话亭
image-gen
AI 生成任意图像
📝
≈ 自动日记
smart-git-commit
智能生成提交信息
🎨
≈ 绘画笔
comic-guide
生成漫画图解（本技能！）
📦 一个 Skill 长什么样？
🤖
哆啦A梦
每个 Skill 就是一个文件夹，里面最重要的是 SKILL.md 文件！ 
# Skill 文件夹结构
my-skill/
├── SKILL.md        # 必需！操作说明书
├── references/     # 可选：参考文档
├── scripts/        # 可选：脚本工具
└── examples/       # 可选：示例
# SKILL.md 长这样 👇
---
name: my-awesome-skill
description: 技能描述...
---

📥 三种安装道具的方式
1
对话安装（最简单）

│

ClawHub = 道具商店
ClawHub 是小龙虾的官方技能商店，已收录 5,400+ 个社区技能。就像哆啦A梦从22世纪的百货公司买道具一样，你可以从 ClawHub 一键安装各种强大的技能！
💡 原来 Skill 就是给 AI 的操作说明书啊！

📝 本话总结

  • Skills 系统 = 百宝袋，每个 Skill 就是一个道具
  • Skill 本质是 SKILL.md 文件，用 Markdown 写的操作说明书
  • 安装方式：对话安装 / ClawHub 商店 / 手动放文件
  • ClawHub 已收录 5,400+ 社区技能，一键安装
