📦 第一步：准备环境
🤖
哆啦A梦
先检查你有没有这些「基础道具」：
🔧 环境要求
  • Node.js v22+ — 小龙虾的运行引擎
  • Python 3.9+ — 部分技能需要
  • Git — 用于克隆代码
  • 2GB+ 内存 — 最低配置
# 检查环境是否就绪
node --version   # 需要 v22+
python3 --version # 需要 3.9+
git --version
🦞 第二步：安装小龙虾
🤖
哆啦A梦
就像在百货公司买道具一样简单，一行命令搞定！ 
# 方式一：npm 全局安装（推荐）
npm install -g openclaw

# 方式二：一键脚本安装
curl -fsSL https://get.openclaw.ai | bash

# 安装完成后验证
openclaw --version
